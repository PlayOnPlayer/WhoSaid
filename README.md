# Cards Against AI

A multiplayer online card game where players answer prompts, then guess which answer was generated by AI.

## How to Play

1. A player creates a room and shares the 4-letter room code
2. Other players join with the code
3. In each round:
   - A random question is shown (20 seconds to answer)
   - Players type their responses
   - AI generates a blended answer
   - All answers are displayed anonymously
   - Players vote on which is AI (30 seconds)
   - Points awarded: +1 for guessing AI correctly, +1 if someone votes for your answer
4. First player to 10 points wins!

## Setup

1. Clone this repository
2. Install dependencies: `npm install`
3. Create a `.env` file with the following:
   ```
   OPENAI_API_KEY=your_openai_api_key_here
   PORT=3000
   NODE_ENV=development
   ```
4. Add your OpenAI API key to `.env`
5. Run the server: `npm start`

## Deployment

### Railway
1. Connect your GitHub repository
2. Add environment variable: `OPENAI_API_KEY`
3. Deploy!

### Render
1. New Web Service
2. Connect repository
3. Build Command: `npm install`
4. Start Command: `npm start`
5. Add environment variable: `OPENAI_API_KEY`

## Development

Run with auto-reload: `npm run dev`

## Tech Stack

- Backend: Node.js + Express + Socket.io
- Frontend: Vanilla HTML/CSS/JavaScript
- AI: OpenAI GPT-3.5-turbo
